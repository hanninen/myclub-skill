# myclub-skill
Skill for AI agents to parse events available in the myClub (myclub.fi) service

## Prerequisites

- `uv` Python package manager installed
- `npm` package manager installed for `npx` command

## Installation

- `uv sync` to install Python dependencies
- `npx playwright install chromium` to have Chromium available for Playwright as a headless browser.
- `npx playwright install-deps` may be needed to get the necessary OS dependencies installed.
